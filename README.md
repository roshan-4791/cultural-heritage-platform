This project focuses on preserving the cultural heritage of local communities by enabling students to collect stories, folk tales, recipes, and traditions from village elders. Collected interviews are organized, subtitled, and showcased through a YouTube channel and digital archive.

## Features
- Upload and organize recorded videos
- Subtitles generation support (YouTube-based)
- Category-wise story listing
- Clean dashboard UI
- Mobile-friendly design
- QR code support for easy access

## Tech Stack
- Next.js 14
- React
- CSS / Tailwind (optional)
- Next.js API Routes
- Vercel Deployment
